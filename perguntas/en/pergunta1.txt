📌 Pergunta:

In SAP Business One, what is the main difference between the "Select Company" and "Create Company" options when accessing the system?

🔘 Alternatives:

A) Select Company allows accessing an existing database, while Create Company generates a new base with initial configurations. ✅

B) Select Company is used to delete a database, while Create Company only views existing reports.

C) Select Company creates users in the system, while Create Company changes the chart of accounts.

D) Select Company configures tax parameters, while Create Company accesses customer data.

E) Select Company is used only by consultants, while Create Company is used by regular users.

F) Select Company allows editing suppliers, while Create Company is used only to change currencies.

✅ Correct response: A

✔ Justification: A
Alternative A is correct because it exactly reflects how the system works:

Select Company → accessing an already existing base for normal use.
Create Company → creating a new database with initial settings (country, currency, chart of accounts, etc.).

❌ Justifications for incorrect alternatives:

B) Justification: ❌ Incorrect
Select Company does not delete data, and Create Company is not just for visualization. Both have completely different functions than those described.

C) Justification: ❌ Incorrect
Creating users and changing the chart of accounts are administrative actions within an existing company, not directly related to selecting or creating a company.

D) Justification: ❌ Incorrect
Tax configuration occurs during the creation or parameterization of the company, not when selecting it. And selecting is not specific for accessing only customers.

E) Justification: ❌ Incorrect
Select Company is used by any user. Create Company is usually done by consultants or IT, not by regular users.

F) Justification: ❌ Incorrect
Editing suppliers and changing currencies are internal company activities; they do not define the function of selecting or creating a company.
